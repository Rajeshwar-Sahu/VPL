☁️ Google Apps Script Database

How it works: A Google Apps Script Web App acts as your serverless database hosted on Google's infrastructure — no Cloud Console, no OAuth, no billing.

Setup (one time):
  1. Open script.google.com → New project
  2. Paste the Apps Script code below into the editor, then click Deploy → New deployment → Web app
  3. Set Execute as: Me | Who has access: Anyone
  4. Copy the Web app URL and paste it below
  5. Admin password is set/checked via the script — change it in the script under ADMIN_PASSWORD
